# Enterprise Network Design & Routing using Cisco Packet Tracer

## Project Overview

This project demonstrates the implementation of an enterprise network using Cisco Packet Tracer.

### Features

- VLAN 10 (HR)
- VLAN 20 (Sales)
- Router-on-a-Stick
- DHCP Configuration
- Extended ACL
- OSPF Dynamic Routing
- Branch Office Connectivity
- End-to-End Connectivity Testing

## Technologies

- Cisco Packet Tracer
- VLAN
- DHCP
- Router-on-a-Stick
- OSPF
- ACL
